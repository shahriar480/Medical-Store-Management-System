# 💊 Medical Store Management System

A desktop-based Medical Store Management System developed using **C# Windows Forms (.NET Framework)** and **Microsoft SQL Server**. The application helps manage medicine inventory, user accounts, billing, sales reports, and staff operations through an intuitive graphical user interface.

---

## 📌 Project Overview

The Medical Store Management System is designed to simplify the day-to-day operations of a pharmacy or medical store. It allows administrators and staff to efficiently manage medicines, process customer bills, maintain inventory, and generate sales reports.

This project was developed as a university project to demonstrate database management, object-oriented programming, and desktop application development using C#.

---

## ✨ Features

### 🔐 Authentication
- Secure login system
- Role-based access (Admin & Staff)

### 👨‍💼 Admin Features
- Dashboard

- Manage users
- View staff information
- Generate sales reports

### 👨‍⚕️ Staff Features

### 📊 Reports

- Billing history

## 🚀 Getting Started

### Prerequisites

- Visual Studio 2019 or later
- .NET Framework
- Microsoft SQL Server
- SQL Server Management Studio (SSMS)

### Installation

1. Clone the repository.

```bash
git clone https://github.com/yourusername/medical-store-management-system.git
```

2. Open the solution in Visual Studio.

3. Import the provided `.bacpac` database into SQL Server.

4. Update the SQL Server connection string if necessary.

5. Build and run the project.
